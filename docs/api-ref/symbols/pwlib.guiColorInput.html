updateColor(color)
Update color. This method allows the change of the color values associated to the current color input.

This method is used by the color picker tool and by the global GUI configChange application event handler.

Parameters:
{Object} color
The new color values. The object must have four properties: red, green, blue and alpha. All values must be between 0 and 1.
